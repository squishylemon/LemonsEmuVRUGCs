Squishylemon's EmuVR UGC

Imac G3

Version: 2.0

This is the iMac G3, now fully interactable. Originally made by War Wolf, it was later converted to a system with macOS 8.1 support. This version also separates the keyboard and mouse. This UGC uses the doscore as there is no official macOS 8.1+ core for RetroArch and is intended to run an emulator called Fusion30. I have written a guide here which you can follow to install macOS 8.1 on this iMac G3.
